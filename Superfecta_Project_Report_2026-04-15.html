Primary conclusion

The project’s prettiest headline is no longer the thing that matters most. The strongest practical evidence still belongs to the Phase 7 OP/CD rule-component basket, not the bigger Phase 8 expansion.

  • Best current holdout: +38.68% on 175 races
  • Recent split: 2024 +0.37% on 109, 2025 +105.38% on 66 — real edge, but not a smooth two-year path
  • Best choice for the main paper-trade lane right now
Validation upgrade

Selector cleanup was worth it

The most defensible quantitative improvement in the whole project came from cleaning the selection logic, not from inventing a new strategy family, but it should still be read as a historical scoring improvement rather than the current frozen deployment benchmark.

  • Historical selector-scoring result improved from +22.46% to +30.42%
  • Current frozen selector benchmark still remains the Train-only yearly selector at +22.46% walk-forward and +14.36% holdout
  • No new rule mining, no fake re-ranking of history
Rule hierarchy

Trust is now easier to explain

The project finally has a clearer answer to the question “which rules do we actually trust?” instead of spreading that answer across ten files. The year split is what makes the hierarchy easier to defend honestly.

  • OP_DURABLE_K7 = safest anchor: mixed holdout, but on the largest OP holdout sample (2024 -47.41% on 68; 2025 +124.61% on 47) plus the strongest walk-forward support (7/10 folds)
  • CD_CORE_K8 = paper-worthy, still useful in the primary paper lane: the steadier current paper candidate because it stayed positive in both holdout years (2024 +45.65% on 41; 2025 +78.21% on 19)
  • OP_REFINED_K7 = interesting, but still a challenger not a replacement: prettier aggregate ROI, but still a smaller mixed-year spike (2024 -25.47% on 33; 2025 +210.02% on 16)

So the real read is not “highest ROI wins.” It is that CD currently looks steadier, OP_DURABLE still has the stronger anchor-grade evidence base, and OP_REFINED still needs more forward sample before it can challenge the anchor seriously.

Rejected hype

Some improvements were real, but not promotable

One of the best things about the current report is that it explicitly rejects seductive but weak stories instead of pretending every metric bump deserves deployment.

  • Harville remains a benchmark, not a live system
  • XGBoost residual correction remains research, not a betting engine
  • Full-data XGBoost retrain commands and RMSE / MAE gains stay in FULL_DATA_RETRAIN_ARTIFACTS.md / validate_full_data_retrain_artifacts.py as model-fit diagnostics only
  • Horse-history features helped prediction, but not enough downstream to change the bet path
